문제 주소
=====================
<https://www.acmicpc.net/problem/2753>

문제 접근 방법
=====================
변수를 정수형으로 저장한다.  
  
두 조건 중에 하나라도 만족하면 윤년이다.
	1. 4의 배수이면서 100의 배수가 아닐때
	2. 400의 배수일때
윤년일때 1 을 출력하고,  
윤년이 아닐때 0 을 출력한다. 


## 코드

```
year = int(input())

if ( year % 4 == 0 and year % 100 != 0 ) or year % 400 == 0:
    print(1)
else:
    print(0)
```
